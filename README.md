## CRUD SPA Using Laravel 9 And Vue.js 3

Simple Create Read Update Delete Single Page Application using Laravel 9 and Vue.js 3

### Tech Stack

- Laravel 9
- Bootstrap 5
- Vue.js 3
- Vue Axios
- Vue Router
- MySQL
- Restful API

### Features

- Read Data
- Create Data (Inc. Validation)
- Update Data (Inc. Validation)
- Delete Data
